## Дальше — System popup (замена battery-слота)

Не входит в исходную таблицу A-E, добавлено 2026-07-19 после того,
как выяснилось: battery-виджет рендерит пустой div на десктопах без
физической батареи — власти профиля питания (Cline №10, бэкенд уже
готов) физически некликабельны. Решение — новая иконка «Система»:
яркость (backlight-сервиса нет, с нуля) + power-profile switch
(бэкенд готов, только UI) + gaming mode (все разом: performance-
профиль + без анимаций/blur + DND + скрыть bar/dock + без screen
tearing). Дизайн-спека — `design.md` §6. Кодовый бриф НЕ написан —
нужна сверка точных `hyprctl`-команд для анимаций/blur/tearing.

## Известные системные ограничения (не менять походя)

- `overflow_y_scroll()` не резолвится в нашем форке gpui — только
  `overflow_hidden()` (клип, не скролл). См. ARCHITECTURE.md §4.1.
- `follow_mouse=1` — популярные попапы никогда не закрываются по потере
  фокуса, только явным действием. См. ARCHITECTURE.md §4.1.
- ydotool на этой машине нестабилен (dual-monitor calibration drift) —
  живые клик-смоки синтетическим вводом ненадёжны, финальную приёмку
  кликом делает пользователь лично.
